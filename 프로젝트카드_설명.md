# 그리드 속 프로젝트 카드, 어떻게 만들어졌나

그리드 안에 들어가는 **개별 프로젝트 카드 한 장**이 어떤 구조와 CSS로 만들어졌는지 정리한 문서입니다.
(그리드 전체 배치 설명은 [GRID_설명.md](GRID_설명.md) 참고)

---

## 1. 카드 한 장의 HTML 구조

카드는 통째로 **링크(`<a>`)**입니다. 클릭하면 프로젝트 페이지로 이동.

```html
<a href="https://example.com" target="_blank" class="project-card">
  <div class="proj-thumb">      <!-- 위: 썸네일 영역 -->
    <!-- <img src="스크린샷 URL" alt="프로젝트 1"> -->
    <span>🌐</span>             <!-- 이미지 없을 때 보이는 이모지 -->
  </div>
  <div class="project-name">프로젝트 이름 1</div>  <!-- 아래: 제목 -->
</a>
```

- 카드 = **썸네일(위) + 이름(아래)** 두 부분
- `<a>` 전체가 클릭 영역 → 카드 아무 데나 눌러도 링크 작동
- `target="_blank"` → 새 탭에서 열림
- 썸네일 안에 `<img>` 주석이 있어서, 주석 풀고 URL 넣으면 실제 스크린샷으로 교체됨

---

## 2. 카드 박스: 반투명 유리 느낌

```css
.project-card {
  background: rgba(255, 255, 255, 0.07);        /* 살짝 흰 반투명 */
  border: 1px solid rgba(255, 255, 255, 0.15);  /* 은은한 테두리 */
  border-radius: 12px;          /* 둥근 모서리 */
  overflow: hidden;             /* 썸네일이 둥근 모서리 밖으로 안 삐져나옴 */
  text-decoration: none;        /* 링크 밑줄 제거 */
  display: flex;
  flex-direction: column;       /* 썸네일 위 / 이름 아래 세로 쌓기 */
  transition: transform 0.25s ease,
              border-color 0.25s ease,
              box-shadow 0.25s ease;
}
```

- `rgba(255,255,255,0.07)` → 어두운 보라 배경이 살짝 비치는 **glassmorphism**(유리) 느낌
- `overflow: hidden` → 안쪽 썸네일 이미지가 카드 둥근 모서리에 맞춰 깔끔하게 잘림
- `flex-direction: column` → 썸네일과 이름이 위아래로 정렬
- `transition`을 **평소 상태**에 미리 걸어둠 → hover 들어갈 때/나올 때 둘 다 부드럽게 움직임

---

## 3. 썸네일 영역: 비율 고정이 핵심

카드마다 썸네일 높이가 다르면 그리드가 지저분해짐. `aspect-ratio`로 **16:9 고정**.

```css
.proj-thumb {
  width: 100%;
  aspect-ratio: 16 / 9;     /* 가로:세로 = 16:9 강제 */
  display: flex;
  align-items: center;       /* 이모지 세로 가운데 */
  justify-content: center;   /* 이모지 가로 가운데 */
  font-size: 30px;           /* 이모지 크기 */
  position: relative;
  overflow: hidden;
}
```

- `aspect-ratio: 16/9` → 카드 너비(185px)에 맞춰 **높이 자동 계산** → 6칸 썸네일 높이가 전부 동일
- `display:flex` + `align/justify center` → 이모지가 정중앙에 옴
- `position: relative` → 나중에 들어올 `<img>`를 절대 위치로 깔기 위한 기준점

---

## 4. 실제 스크린샷 넣을 때 (img)

썸네일에 이미지를 넣으면 비율 안 깨지고 꽉 채워짐.

```css
.proj-thumb img {
  position: absolute;
  inset: 0;                 /* top/right/bottom/left 0 → 영역 꽉 채움 */
  width: 100%;
  height: 100%;
  object-fit: cover;        /* 비율 유지하며 잘라서 채우기 */
}
```

- `position:absolute` + `inset:0` → 썸네일 영역을 완전히 덮음
- `object-fit: cover` → 가로세로 비율 안 깨지고, 넘치는 부분만 잘림

---

## 5. 카드마다 다른 색 (nth-child 그라데이션)

썸네일이 비어 있어도 심심하지 않게, 6개 카드에 **각각 다른 그라데이션**.

```css
.project-card:nth-child(1) .proj-thumb { background: linear-gradient(135deg, #1a1a4e, #3a3080); }
.project-card:nth-child(2) .proj-thumb { background: linear-gradient(135deg, #0f3460, #1a6090); }
.project-card:nth-child(3) .proj-thumb { background: linear-gradient(135deg, #2d1b69, #6a3fa0); }
.project-card:nth-child(4) .proj-thumb { background: linear-gradient(135deg, #0a3d2e, #0d8060); }
.project-card:nth-child(5) .proj-thumb { background: linear-gradient(135deg, #4a1530, #a0204a); }
.project-card:nth-child(6) .proj-thumb { background: linear-gradient(135deg, #102840, #1a5080); }
```

- `:nth-child(n)` → **몇 번째 카드인지**로 골라 색 지정
- `linear-gradient(135deg, ...)` → 대각선 그라데이션이라 입체적
- 전부 어두운 톤 → 배경과 따로 놀지 않으면서 칸마다 개성

## 6. 카드 이름(제목)

```css
.project-name {
  padding: 10px 10px;
  color: rgba(255, 255, 255, 0.88);   /* 거의 흰색 (살짝 투명) */
  font-size: 13px;
  font-weight: 500;
  text-align: center;
  font-family: "Noto Sans KR", sans-serif;
  line-height: 1.4;
}
```

- `padding` → 글자가 카드 벽에 안 붙음
- `rgba(...0.88)` → 완전 흰색보다 살짝 부드러운 흰색
- `text-align: center` → 제목 가운데 정렬

---

## 7. 마우스 올리면: 둥실 떠오르는 효과

카드를 "이쁘게" 만드는 가장 큰 한 방.

```css
.project-card:hover {
  transform: translateY(-6px);                      /* 위로 6px 떠오름 */
  border-color: rgba(16, 185, 129, 0.7);            /* 초록 테두리 강조 */
  box-shadow: 0 10px 28px rgba(16, 185, 129, 0.2);  /* 초록 그림자 */
}
```

- `translateY(-6px)` → 카드가 **위로 둥실** 뜸
- 테두리·그림자가 초록(`#10b981`)으로 → 프로필 사진 테두리·직함 색과 통일
- 2번의 `transition` 덕분에 0.25초 동안 부드럽게 전환

---

## 한 줄 요약

> **카드 전체를 `<a>` 링크로 → 반투명 박스(`rgba`+`border-radius`+`overflow:hidden`) → 썸네일은 `aspect-ratio:16/9`로 높이 통일 → `nth-child`로 카드마다 색 → `:hover`로 위로 띄우고 초록 강조.**

이 다섯 가지가 모여 그리드 속 카드 한 장이 완성됩니다.
