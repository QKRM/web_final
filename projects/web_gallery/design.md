## 📋 심플 디지털 음식 메뉴판 UI 명세서 (HTML/CSS 전용)

### 1. 프로젝트 개요

* **목적:** 매장 테이블에 비치하는 태블릿 화면용 정적(Static) 메뉴판 레이아웃 구현. (순수 HTML/CSS 사용)
* **디자인 컨셉:** 사진을 강조하는 심플하고 모던한 카드형 UI.

---

### 2. 레이아웃 구조 (Layout)

| 섹션          | 설명                                                 | 비고            |
| ------------- | ---------------------------------------------------- | --------------- |
| **Header**    | 매장 로고(또는 매장명)와 간단한 소개 문구 중앙 배치. | 상단 여백 확보  |
| **Menu Grid** | 음식 사진과 정보가 담긴 카드들이 나열되는 메인 영역. | 그리드 레이아웃 |
| **Footer**    | 매장 주소, 영업시간, 연락처 등 하단 정보.            | 하단 중앙 정렬  |

---

### 3. 컴포넌트 상세 명세 (Component Details)

#### A. 헤더 영역 (Header Area)

- **배치:** 화면 상단 중앙 정렬 (`text-align: center`).
- **내용:** 큰 글씨의 매장 이름, 그 아래 작은 글씨의 슬로건(예: "매일 아침 신선한 재료로 요리합니다").
- **디자인:** 하단에 연한 구분선(`border-bottom`)을 그어 메인 영역과 분리.

#### B. 메뉴 카드 (Menu Card)

각각의 메뉴를 담는 개별 박스(Box)입니다.

- **구조:** 상단(이미지) + 하단(텍스트 정보)
- **이미지 영역:**
- 비율: 4:3 또는 1:1 비율을 유지 (`aspect-ratio` 또는 높이 고정).
- 스타일: 이미지가 카드를 가득 채우며 잘리거나 찌그러지지 않아야 함 (`object-fit: cover`).

- **정보 영역 (텍스트):**
- **메뉴명:** 굵게, 좌측 정렬.
- **가격:** 메뉴명과 같은 줄의 우측 정렬 또는 메뉴명 아래 배치. (Flexbox 활용 추천)
- **설명(Description):** 메뉴명 아래에 작고 흐린 글씨로 1~2줄 추가 (예: "매콤한 특제 소스가 들어간 시그니처 덮밥").

- **디자인 요소:**
- 부드러운 테두리 둥글기 (`border-radius`).
- 배경색과 구분되는 미세한 그림자 효과 (`box-shadow`).

- **인터랙션:** 마우스 호버 시 카드가 살짝 떠오르는 느낌의 부드러운 애니메이션 효과 (`transition`, `transform`).

---

### 4. 디자인 가이드 (CSS Guide)

#### **Color Palette**

- **배경 (Body):** `#FAFAFA` (아주 연한 회색)
- **카드 배경:** `#FFFFFF` (순백색)
- **텍스트 (제목/메뉴명):** `#333333` (진한 회색)
- **텍스트 (설명/가격):** `#888888` (중간 회색)
- **포인트 색상:** (선택 사항) 강조하고 싶은 부분(예: 가격)에 사용할 색상 (예: `#E53935` 붉은색 계열)

#### **Typography**

- **웹 폰트:** 맑은 고딕, Noto Sans KR 등 깔끔한 산세리프(Sans-serif) 폰트 사용.
- **강조:** `font-weight` 속성을 활용하여 메뉴명은 두껍게, 설명은 얇게 처리.

---

### 5. 반응형 요구사항 (Responsive Grid)

화면 크기(Viewport)에 따라 메뉴 카드의 배열을 변경합니다.

- **데스크톱 (큰 화면):** 한 줄에 **4개**씩 카드 배치.
- **태블릿 (중간 화면):** 한 줄에 **3개**씩 또는 **2개**씩 카드 배치.
- **모바일 (작은 화면):** 한 줄에 **1개**씩 카드 배치 (카드가 세로로 꽉 차게 나열됨).

---

### 6. 구현 시 핵심 포인트 (Key Learning Points)

- **CSS Grid (추천) 또는 Flexbox:** 메뉴 카드들을 화면 너비에 맞게 규칙적으로 나열하는 방법 숙달.
- **Box Model:** `padding`과 `margin`을 적절히 사용하여 카드 안팎의 여백 조절.
- **Media Query:** `@media` 룰을 사용하여 화면 크기별 레이아웃 변화 적용.
