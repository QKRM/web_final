# 카드에 프로젝트 넣는 법 (간단 가이드)

그리드 속 카드에 **새 프로젝트**를 연결하는 방법.

---

## 1. 프로젝트 파일 위치

프로젝트 폴더를 `projects/` 안에 넣는다.

```
web_namecard/
├─ index.html
├─ style.css
└─ projects/
   ├─ web_animation/
   │   └─ index.html      ← 이게 카드 클릭 시 열림
   ├─ web_gallery/
   │   └─ index.html
   └─ web_새프로젝트/
       └─ index.html
```

**주의**: 폴더 안에 `.git`이 있으면 삭제. 안 그러면 git이 submodule로 인식해서 푸쉬 시 빈 폴더로 올라감.

```bash
rm -rf projects/web_새프로젝트/.git
```

---

## 2. `index.html`에 카드 추가

`.projects-grid` 안에 카드 1장 복사 + 수정:

```html
<a href="projects/web_새프로젝트/index.html" target="_blank" class="project-card">
  <div class="proj-thumb">
    <span>🎮</span>                       <!-- 이모지 (또는 <img>) -->
  </div>
  <div class="project-name">새 프로젝트 이름</div>
</a>
```

바꿀 3가지:
| 항목 | 위치 | 예시 |
|---|---|---|
| 링크 경로 | `href="..."` | `projects/web_새프로젝트/index.html` |
| 아이콘 | `<span>이모지</span>` | `🎮` `🎵` `🧮` 등 |
| 카드 이름 | `<div class="project-name">` | `게임`, `음악 플레이어` |

---

## 3. 카드 색 (선택)

`style.css` 맨 아래에 `nth-child(N)` 한 줄 추가. N = 카드 순번.

```css
.project-card:nth-child(8) .proj-thumb { background: linear-gradient(135deg, #2a1a4e, #4a2080); }
```

색 안 정하면 기본 배경(투명)으로 나옴 → 추가 권장.

---

## 4. 실제 스크린샷 쓰고 싶을 때

이모지 대신 이미지:

```html
<div class="proj-thumb">
  <img src="projects/web_새프로젝트/screenshot.png" alt="새 프로젝트">
</div>
```

CSS에 이미 `object-fit: cover` 설정돼있어서 **비율 안 깨지고 꽉 채워짐**. 가로세로 비율은 16:9 권장 (그리드가 16:9로 고정).

---

## 5. 푸쉬

```bash
git add -A
git commit -m "Add 새 프로젝트 card"
git push
```

GitHub Pages 켜놨으면 자동 반영.

---

## 한 줄 요약

> **`projects/`에 폴더 복사 (.git 제거) → `index.html`에 `<a class="project-card">` 한 장 추가 → href·이모지·이름 3개만 수정 → 푸쉬.**
